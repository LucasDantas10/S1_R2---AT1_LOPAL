Este repositório reúne uma série de exercícios desenvolvidos no Flowgorithm, com o objetivo de praticar conceitos básicos de lógica de programação por meio de fluxogramas.

Exercício Conta: Fluxograma que realiza a leitura de dois números, efetua o cálculo da soma entre eles e exibe o resultado obtido de forma clara.

Exercício Idade: Fluxograma que solicita a idade de uma pessoa e, com base nesse dado, informa se ela é maior ou menor de idade.

Exercício Nome: Fluxograma que lê o nome de uma pessoa e apresenta uma mensagem de saudação personalizada, como “Olá, [nome]!”.

Exercício Nota: Fluxograma responsável por calcular a média a partir de duas notas informadas pelo usuário, exibindo o resultado final.

Exercícios Números: Fluxograma que lê um número e identifica se ele é positivo, negativo ou igual a zero, informando o resultado ao usuário.

Exercício Revisão: Fluxograma de revisão que reúne e reforça os conceitos trabalhados nos exercícios anteriores.
