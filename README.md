Este repositório contem exercicios realizados no Flowgorithm:
ExercicioConta: Fluxograma que le dois números, calcula e exibe o resultado da soma.
ExercicioIdade: Fluxograma que pergunta a idade de uma pessoa e mostra se ela é maior ou menor de idade.
ExercicioNome: FLuxograma que Le o nome de uma pessoa e digita "Olá, tal pessoa!".
ExercicioNota: Fluxograma que calcula a média a partir de duas notas.
ExerciciosNumeros: Fluxograma que le um número e informa se ele é positivo, negativo ou zero
exerciciorevisão: Fluxograma de revisão
